ADD CSS

There are three ways:

  • External
  • internal
  • Inline

    • Note: an inline style has the highest priority, and will override external and internal styles.

    External CSS

  • It is used when you want to define style for multiple pages with external style sheet you can shange the look of entire web page by changing just one file.
      • It is define within the link> element under the head section.

    Must be saved with .css extention

    Internal CSS

  • internal CSS is used for the unique style of single html page
      • It define inside the style> within the head section

    Inline CSS

  • It is use for the style of single element
      • To use inline css style atribute is added in the relevant element, It contain any CSS property.
